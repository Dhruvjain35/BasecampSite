Ages

Eight through fourteen

Three tracks by age. Same destination, different pace.

Prerequisites

None required

We start from zero in Explorer. Builder & Creator pick up the pace.

Outcome

Working app on the internet, GitHub repo, and a short demo. Yours forever.

The six weeks

From zero to shipped.

Each week builds on the last. By week six, they're presenting on stage.

1

Foundations

Set up VS Code + Claude Code Pro, learn Python basics, write a first working script.

2

How AI works

What a model is, in plain words. See one in action and start dreaming.

3

First AI app

Call the OpenAI API. Build a chatbot or story generator in one session.

4

Project sprint

Pick a capstone. Sketch it. Build it. Code reviews every session.

5

Going live

Push to GitHub. Deploy with one command. Get a real public URL.

6

Demo Day

Five-minute presentation to parents, mentors, and friends. The grand finale.

Each session

Two hours · Three parts.

0:00 – 0:30

Learn

One concept, taught with live coding. No slides. Kids follow along.

0:30 – 1:30

Build

Hands-on time with mentors. They code, hit bugs, fix them.

1:30 – 2:00

Share

Each kid shows what they made. Quick demo. Practice for Demo Day.
