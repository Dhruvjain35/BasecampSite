TEAM

Built by students.

High schoolers in Frisco who learned this stuff ourselves. We teach the way we wish we'd been taught.

We started Basecamp because every kids' coding camp looked the same. Drag-and-drop, cartoon tutorials, no connection to how real software gets built.

We're juniors at our high school in Frisco, TX. We've shipped real apps, published research, and competed nationally. When we learned AI, no one handed us a baby version — we used the real tools. We're bringing that same approach to younger kids, at their pace.

What we believe

Four principles.

1

Real tools only

If working engineers don't use it, we don't teach it. Python and VS Code — not Scratch.

2

Ship or it didn't happen

Every project gets a real URL. Code that only runs on your laptop isn't a thing yet.

3

Struggle is the curriculum

We hand kids problems and docs. The debugging is where the learning lives.

4

Age isn't a limit

An 8-year-old can call an API. A 12-year-old can ship a web app. We've watched it happen.
